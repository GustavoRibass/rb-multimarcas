# Projeto RB Multimarcas

Projeto desenvolvido para o Módulo 2 – disciplina de Experiência do Usuário.

## Tecnologias utilizadas
- HTML5
- CSS3 (com Bootstrap)
- JavaScript
- Arrays para geração dinâmica de conteúdo
- Validação de formulário com JavaScript

## Estrutura do projeto
- index.html – Página inicial
- estoque.html – Estoque dinâmico
- contato.html – Formulário validado
- quemsomos.html – Informações da empresa
